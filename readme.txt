# TP note sujet 1
***

Le tp porte sur des logements. L'objectif final est de prédire le prix d'un logement x en fonction des logements environnants. Le programme doit prendre en compte différentes notions pour établir une "distance" entre les différents logements.

## Technologies
***

Liste des technologies utilisées:
* <stdio.h>
* <stdlib.h>
* <math.h>

## Utilisation
***

Il faut que les fichiers suivants se trouvent dans un même dossier:
* sujet1.c
* sujet1.h
* airbnb_donnees_propre.csv
* Makefile

Dans le terminal, se placer dans la direction correspondante au dossier du sujet2: 
```
make
./prog

```
